ASCOM.NGCAT.Focuser Architecture

RoboFocus ASCOM v2 driver — C# .NET Framework 4.8

Driver

  • • Implements ASCOM IFocuserV2
  • • C# .NET Framework 4.8
  • • COM-visible DLL
  • • Absolute positioning (0–10,000 steps)

Protocol

  • • RoboFocus serial protocol
  • • 9-byte messages with checksum
  • • 9600 baud, RS-232
  • • FG/FT/FV/FQ/FP/FB/FL/FC/FS

Deployment

  • • Inno Setup installer
  • • COM registration (32/64-bit)
  • • ASCOM Platform 6.2+ required
  • • GPL v3 license
